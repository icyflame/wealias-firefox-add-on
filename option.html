Type '_add alias' in the inputbox/URL bar to add a new alias bound to the current URL you are.
In order to use this feature, the tabs permission needs to be granted.
If you tick the option you will be asked for grant this permission, and you can revoke it anytime you want unticking it.

Show help

Each entry of the list has format alias≺space≻URL
Type alias text into the input box and the current tab will be updated with corresponding URL.
> `1, `2, etc. replaces the word digitth of text on URL.
> `s is replaced with any leftover text parts unmatched by `digit placeholders.
> If `s is not used, any leftover text parts are appended to resulting URL.
The "*" is the default alias for any un-matched text.
